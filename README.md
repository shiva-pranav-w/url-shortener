# URL Shortener

Production-ready URL shortener with:

- JWT Authentication
- Redis caching
- Rate limiting
- Async hit counting
- PostgreSQL persistence
- Deployed on Render

## Live API
https://url-shortener-x6c5.onrender.com

## Features
- Shorten URLs
- Custom short codes
- Analytics (hit count)
- Top URLs per user

## Tech Stack
- Java + Spring Boot
- PostgreSQL (Neon)
- Redis (Upstash)
- Docker
- Render (deployment)
